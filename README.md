# Food Price Intelligence Dashboard (Nigeria)

## Overview
The Food Price Intelligence Dashboard is a real-time monitoring platform that tracks food prices across Nigerian markets. 

The system provides interactive data visualization, price comparison between markets, inflation monitoring, and predictive analytics to support better food market insights.

## Key Features

• Real-time food price monitoring  
• Market price comparison across regions  
• Nigeria market location map  
• Price forecasting using machine learning  
• Inflation alert system  
• AI-based anomaly detection for suspicious price reports  
• Secure market price submission form  
• Secure dataset download

## Technology Stack

Python  
Streamlit  
Pandas  
Plotly  
Scikit-learn  

## How to Run the Project

1. Clone the repository
git clone https://github.com/salimbello1025/marketpulse-dashboard.git
2. Install required libraries
 pip install -r requirements.txt
3. Run the dashboard
streamlit run app.py
